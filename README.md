# Github Profile Viewer

GitHub user's profile viewer using GitHub API. Display information, repositries.  
Built with React.

## Demo

[View Demo](https://chocolat5.github.io/react-github-profile-viewer/)

## Getting Started

1. Clone this repo

```
git clone git@github.com:chocolat5/react-github-profile-viewer.git <PROJECT_NAME>
```

2. Move to the appropriate directory

```
cd <PROJECT_NAME>
```

3. Install npm package

```
npm install --save-dev
```

4. Run the project

```
npm run start
```
